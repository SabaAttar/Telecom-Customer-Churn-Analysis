# 📊 Telecom Customer Churn Analysis (Power BI)

## 📌 Project Overview
This project analyzes telecom customer churn data using Power BI to identify key factors influencing customer attrition and its impact on revenue. The dashboard provides insights into customer behavior, churn patterns, and high-risk segments.

---

## 🎯 Objectives
- Analyze customer churn behavior  
- Identify key drivers of churn  
- Evaluate revenue loss due to churn  
- Segment high-risk customers  
- Provide data-driven insights for retention strategies  

---

## 📂 Dataset Source
The dataset used in this project is publicly available and sourced from GitHub/Kaggle.

It contains telecom customer data including demographics, services, tenure, and churn information.

---

## 📊 Dashboard Pages

### 1️⃣ Churn Overview
- Total Customers, Churn Customers, Churn Rate  
- Churn by Contract Type  
- Churn by Tenure  
- Internet Service vs Churn  
- Payment Method vs Churn  
- Monthly Charges vs Churn  

---

### 2️⃣ Customer Demographics Analysis
- Churn by Gender  
- Churn by Age Group  
- Churn by Senior Citizen  
- Churn by Marital Status  
- Churn by Dependents  

---

### 3️⃣ Revenue Impact & Customer Churn Risk
- Revenue Lost due to Churn  
- Average Revenue per User (ARPU)  
- Churn Revenue Percentage  
- Revenue by Tenure Group  
- Revenue Loss by Contract Type  
- Top Churn Reasons  

---

## 🔍 Key Insights
- Month-to-month contract customers have the highest churn  
- Customers in early tenure (0–12 months) are more likely to churn  
- Fiber optic users show higher churn rates  
- Customers using mailed check have higher churn  
- Higher monthly charges are associated with increased churn  
- Gender has no significant impact on churn
