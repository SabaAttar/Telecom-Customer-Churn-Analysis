# 📊 Telecom Customer Churn Analysis

## 📌 Problem Statement
This project analyzes telecom customer data to identify key factors contributing to customer churn and provide actionable insights to improve customer retention. The objective is to understand customer behavior, detect patterns leading to churn, and help businesses take data-driven decisions to reduce customer loss.

---

## 📂 Dataset
- **Source:** IBM Telco Customer Dataset  
- **Total Records:** 7,000+ customers  
- **Description:** Dataset includes customer demographics, service details, account information, and churn status.

---

## 🛠 Tools & Technologies
- **Power BI** – Data cleaning, transformation, and dashboard creation  

---

## 📊 Key Analysis Performed
- Calculated overall **churn rate**
- Analyzed churn based on **contract type**
- Evaluated churn trends across **internet services**
- Performed **customer segmentation** based on tenure
- Analyzed relationship between **monthly charges and churn**
- Identified patterns in **payment methods and churn behavior**

---

## ⚠️ Challenges Faced & Solutions

### 1️⃣ Data Cleaning Issues
- **Problem:** Some columns like *TotalCharges* had missing or inconsistent values  
- **Solution:** Handled missing values and corrected data types using Power BI transformations  

---

### 2️⃣ Data Type Mismatch
- **Problem:** Certain numeric columns were stored as text  
- **Solution:** Converted them into proper numeric format in Power Query  

---

### 3️⃣ Understanding Business Context
- **Problem:** Raw data did not directly explain why customers churn  
- **Solution:** Created calculated columns and segmented data (e.g., tenure groups) to derive meaningful insights  

---

### 4️⃣ Identifying Key Drivers of Churn
- **Problem:** Multiple factors affected churn, making it difficult to identify the main reasons  
- **Solution:** Used visual analysis and comparisons (contract type, charges, tenure) to identify key churn drivers

## 🔍 Key Insights
- Approximately **26% of customers have churned**
- Customers with **month-to-month contracts show the highest churn rate**
- Higher **monthly charges are strongly correlated with churn**
- Customers with **shorter tenure (new customers)** are more likely to churn
- **Fiber optic users** show higher churn compared to DSL users

---

## 💡 Business Recommendations
- Encourage customers to switch to **long-term contracts**
- Provide **discounts or offers** for high monthly charge customers
- Improve service quality for **high-risk customer segments**
- Focus retention strategies on **new customers (low tenure)**

---

## 🚀 Business Impact
This project simulates a real-world telecom scenario where businesses can reduce customer churn by identifying high-risk customers and implementing targeted retention strategies.
